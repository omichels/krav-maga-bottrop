# Krav Maga Bottrop - Website

Moderne Webseite für den Krav Maga und Selbstverteidigungsverein in Bottrop.

## Features

- Responsive Design (mobile-first)
- Moderne Benutzeroberfläche
- Interaktive Navigation
- Kontaktformular
- Trainingszeiten-Übersicht
- Smooth Scrolling
- Animationen

## Hosting

Diese Webseite wird über GitHub Pages gehostet.

## Anpassungen

Um die Webseite anzupassen, bearbeite folgende Dateien:

- `index.html` - Hauptinhalt und Struktur
- `style.css` - Styling und Farben
- `script.js` - Interaktive Funktionen

### Wichtige Anpassungen:

1. **Kontaktdaten**: In `index.html` im Kontakt-Bereich die Adresse, Email und Telefonnummer anpassen
2. **Trainingszeiten**: Die Zeiten im Schedule-Bereich anpassen
3. **Farben**: In `style.css` die CSS-Variablen unter `:root` ändern

## Lokale Entwicklung

Öffne einfach die `index.html` in deinem Browser oder starte einen lokalen Webserver:

```bash
python -m http.server 8000
```

Dann öffne `http://localhost:8000` im Browser.
